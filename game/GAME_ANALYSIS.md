## 二、升级学招系统详情（参考绿宝石）

### 2.1 机制说明

- 每只宝可梦拥有 `levelMoves` 数组，定义各等级可学的技能
- **Lv.1 学基础攻击，Lv.4 学变化技**（保证初始 Lv.5 有2个技能，参考绿宝石）
- 升级时自动检查是否有新技能可学
- 已有4个技能时，自动替换最早学会的技能（队列式替换）
- `getLearnedMoves()` 根据当前等级筛选可用技能，取等级最高的4个（自动去重）
- 进化型在 Lv.1 继承前置形态的招式（如喷火龙 Lv.1 有撞击+火花+火焰放射）

### 2.2 学招设计原则（参考绿宝石）

| 阶段 | 等级 | 技能类型 | 示例 |
|------|------|---------|------|
| 初始 | Lv.1 | 基础攻击（一般系） | 撞击、抓 |
| 早期 | Lv.4 | 变化技（降攻/降防） | 叫声、瞪眼 |
| 初级 | Lv.8 | 本系初级攻击 | 火花、水枪、藤鞭 |
| 中级 | Lv.13-14 | 本系中级攻击 | 飞叶快刀、冲浪 |
| 进化 | Lv.18-20 | 进化后新招/副属性 | 水之波动、吸收 |
| 高级 | Lv.23-30 | 本系高级攻击 | 火焰放射、阳光烈焰 |
| 终极 | Lv.38-48 | 本系最强招 | 大字爆炎、加农水炮、日光束 |

## 三、区域等级缩放

### 3.1 等级公式

```
minLevel = max(2, floor(tier² × 0.8) + 1)
maxLevel = floor(tier² × 1.5) + 3
```

### 3.2 各Tier等级范围

| Tier | 等级范围 | 代表区域 | 难度 |
|------|---------|---------|------|
| Tier 1 | Lv.2 ~ Lv.4 | 新手镇、1号道路 | 入门 |
| Tier 2 | Lv.4 ~ Lv.9 | 常青森林、2号道路 | 初级 |
| Tier 3 | Lv.8 ~ Lv.16 | 月见山、华蓝市 | 中级 |
| Tier 4 | Lv.13 ~ Lv.27 | 岩石隧道、红莲废墟 | 高级 |
| Tier 5 | Lv.21 ~ Lv.40 | 石英高原、狩猎区 | 终极 |

### 3.3 说明

- 等级基于**宝可梦自身的Tier**计算，而非区域Tier
- 公式采用二次函数，前期等级增长缓慢（贴近绿宝石的 Lv.2-4 起步），后期加速
- Tier 3-4区域中，Tier 3宝可梦Lv.8-16，Tier 4宝可梦Lv.13-27
- 这保证了同区域内也有等级差异，增加战斗策略性

## 四、倒下切换机制

### 4.1 流程

1. 宝可梦HP降为0时，检查队伍中是否有存活宝可梦
2. 有存活 → 显示选择界面，列出所有存活宝可梦
3. 玩家选择下一只宝可梦 → 继续战斗
4. 野生战斗可选择逃跑（选项0）
5. 训练家战不可逃跑，必须选择下一只
6. 全部倒下 → 战斗失败

### 4.2 经验分配

- 战斗胜利后，经验平均分配给所有存活的宝可梦
- 公式: `每只获得 = 总经验 / 存活数量`
